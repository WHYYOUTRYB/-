# 个人博客 · 完整执行计划

> 一个以**学业笔记**为核心的个人知识站点。笔记按"学科 → 章节 → 笔记"三级组织，是站点主角；随笔、作品集作为次要补充。
>
> 面向中文读者，极简文字风格，阅读优先。

---

## 技术栈

| 层 | 选型 | 理由 |
|----|------|------|
| 框架 | **Astro 6** | 内容网站首选，默认零 JS，Content Collections 强类型 |
| 样式 | **Tailwind CSS 4.2.1** | 原子化，极简排版易实现（注：必须锁定 4.2.1 以对齐 Astro 的 vite 7） |
| 公式 | remark-math + rehype-katex + KaTeX | Obsidian 语法兼容（`$...$` / `$$...$$`） |
| 链接 | remark-wiki-link | 支持 `[[Wiki链接]]`，Obsidian 兼容 |
| 代码 | Shiki（Astro 内置） | 亮暗双主题语法高亮 |
| 搜索 | **Pagefind** | 静态索引，零后端，支持中文 |
| SEO | @astrojs/sitemap + @astrojs/rss | 站点地图与订阅 |
| 部署 | **Vercel** | git push 自动部署 + 预览分支 |

## 信息架构（核心）

### 三级层级

```
学科 (Subject)   配置驱动，可扩展，URL slug 唯一
  └── 章节 (Chapter)   笔记 frontmatter 字段，动态聚合
       └── 笔记 (Note)   Markdown 文件
```

### 笔记 frontmatter

```yaml
---
title: 极限的定义
subject: advanced-math        # 必填，对应 subjects.config.ts 的 slug
chapter: 第一章 极限与连续     # 必填，学科页据此动态分组
order: 1                      # 可选，章节内排序
pubDate: 2026-03-01           # 必填
description: 数列极限的 ε-N 定义
tags: [极限, 定义]             # 必填
draft: false                  # 可选
---
```

URL：`/notes/[subject]/YYYY-MM-DD-slug`

### 学科管理

学科不写死，由 `src/content/subjects.config.ts` 驱动。**加学科 = 加一行配置**。

```ts
export const subjects: Subject[] = [
  { slug: "advanced-math", name: "高等数学", icon: "∫", order: 1 },
  // ...
]
```

```

```

## 视觉系统

- **字体：** 思源宋体（中文衬线）+ 系统衬线英文 + KaTeX Computer Modern（公式）+ JetBrains Mono（代码）
- **色彩：** 暖白底/深灰文（浅色），深底浅文（深色），单一墨蓝强调色用于链接
- **排版：** 正文宽 680px，行高 1.7–1.8，大量留白，纯文字无装饰
- **主题切换：** localStorage 记忆 + 首次跟随系统 + 无闪烁（内联预设脚本）
